# Yankr

Safely stop USB activity so you can unplug your drive without manually ejecting it every time.

## Features
- One-click stop
- Lightweight
- Native macOS app

## Installation
Download the latest `Yankr.dmg` from the Releases page and drag Yankr into Applications.
